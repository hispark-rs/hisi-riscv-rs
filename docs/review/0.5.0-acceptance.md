# hisi-riscv-hal 0.5.0 验收标准（Definition of Done）

> 日期：2026-06-15 · 范围：0.5.0 这一波 breaking（config 收紧 + API 风格清债 + DMA 完整 + 中断路由 + drop-to-disable + chip 去默认）。

## A. 类型化配置收紧（每驱动）— Hard gate

**总不变量**：config 面不接受任何"能写出却被静默 clamp/截断/没接时钟"的值；越界在构造点返回 `None`，不静默处理。

- [x] **pwm** ✅ 已完成（`PwmPeriod`/`Duty` + 自起时钟，HIL `pwm_configure_and_enable` 绿）。
- [x] **spi** ✅（`SpiHz`/`DataBits` newtype，`SpiError::Overflow→Overrun`，host 边界测试；HIL `spi0_loopback` 绿）。
- [x] **i2c** ✅（`Speed{Standard,Fast}` 对齐 BS2X，`new_i2c0/1(speed)`，host `speed_hz_values`；HIL `i2c0_scl_config` 绿）。
- [x] **uart** ✅（`BaudRate` newtype `try_new` 拒绝低端 clamp / 高端 div=0，`configure_uart` 去 clamp，host `baud_rate_rejects_out_of_range`+proptest；HIL `uart0_divider_config` 用 `baud()`）。
- [x] **i2s** ✅（type-state `new_master(&MasterConfig)`/`new_slave(&SlaveConfig)`，分频按 vendor 从 `data_width·channels` 派生→零分频 Master 不可表达 + 全 5×4 组合派生分频在域内[host `derived_dividers_in_range`]；修正 mode/crg/data_width 寄存器位 + 重制 DataWidth{16/18/20/24/32}/ChannelCount{2/4/8/16} 为硅枚举；自起 CKEN(11/12)+CMU。HIL `i2s_version_live` 改 `new_master` 且 embedded-test 编译通过。板上无 codec→主时序仅寄存器复验）。
- [x] **wdt** ✅（`WdtTimeout::from_ms`→`Option`（拒绝 0 / >MAX_MS，不静默饱和），`configure(WdtTimeout,…)->Result<(),WdtError>`，`request_counter` 有界轮询→`Busy`，`counter_value->Result`；`WdtError` non_exhaustive+defmt。host 13 测；HIL `wdt_configure_saturates_load` 改为「拒绝 300s + 精确编程 1s load」；callers bs21/bs20 已改）。
- [x] **timer** ✅（`start_micros`/`start_millis`→`Result<(),TimerError>`（`try_ticks_for_us/ms` u64 数学 + 越界 `TicksOverflow`，不再静默 clamp 到 u32::MAX）；`DelayNs` trait 保留饱和语义（无错误通道）；`current_value` 轮询本已有界。host 溢出测试边界精确；HIL `timer_counter_advances` 不受影响）。
- [x] **rtc / gadc / lsadc** ✅（gadc `convert_once` 无界轮询→有界(`GADC_DONE_POLL_LIMIT`)+`read()->Result<i32,GadcError>`+AFE 前提 doc；lsadc `sample_cnt`/`cast_cnt`/`rxintsize` 静默掩码→`SampleCount`/`CastCount`/`FifoWaterline` newtype 构造拒绝；rtc(v100+v150) 加 32.768kHz 晶振前提 doc，v150 读已是有界 `for` 轮询，ws63 rtc 无 poll。grep 确认这三个驱动无 `while …{}` 无界轮询；rtc/gadc 不上板已文档说明）。
- [x] **全驱动 scan** ✅（`scan.sh` 跑全 src：sfc 64B 写越界静默截断→`SfcError::BufferTooLong` 拒绝；timer/rtc/rtc_v150 raw `load`、spi 常量分频 clamp 均加豁免注释（全宽寄存器/编译期常量，非用户面）；其余命中为 operational embedded-hal clamp / 时间算术 / busy-delay，非 config 面静默 clamp。306 host 测通过）。

## B. API 风格清债 — Hard gate（除注明 soft）

- [x] **错误处理** ✅（`*Error`（含 0.5.0 新增 `WdtError`/`TimerError`/`GadcError`/`SfcError::BufferTooLong`）全 `#[non_exhaustive]`；`SpiError::Overflow→ErrorKind::Overrun`；`i2c_v151::I2cError` 接 `embedded_hal::i2c::Error`）。
- [x] **defmt** ✅（`defmt` optional feature + 错误/config enum 派生 `#[cfg_attr(feature="defmt", derive(defmt::Format))]`；CI matrix 已含 defmt job）。
- [x] **文档强制** ✅（`#![warn(missing_docs)]` 开；ws63 backfill 至 **0**（16-agent workflow + i2c.rs 手补），bs2x-only 模块 backfill（gadc/usb/rtc_v150/i2c_v151/keyscan/trng/qdec/pdm）；`get_gpio_mux`→`gpio_mux` 已改。**待**：backfill 全绿后翻 `deny` + `# Safety`/`# Errors` 抽查。
- [x] **所有权** ✅（`peripheral!` 宏 `reborrow(&mut self)` 已加）。
- [x] **trait/sealing** ✅（`private.rs` 残留 `PeripheralInput/Output` marker 已删；GPIO `Sealed` 收敛到 `Output/Input/Flex`）。
- [x] **MSRV** ✅（`rust-version = "1.85"` 在 hal Cargo.toml 声明 + crate doc 一行政策；无 hal README 故政策入 crate doc）。
- [x] **GPIO 双 API** ✅（遗留 `GpioPin<MODE>` **直接删除**（比 deprecate 更彻底，0.5.0 breaking）；`Flex::set_as_input/set_as_output` 显式方向；`Input/Output/Flex::degrade()->AnyPin`（Output 用 ManuallyDrop 不触发 safe-state Drop）。gpio HIL 句柄在作用域内不受影响。**soft 残留**：`ulp_gpio` 命名统一延后（低价值，ulp 是独立 ULP-GPIO 子系统，命名已自洽）。
- [x] **构造清债**（部分）：裸标量→newtype 已全面落实（`SpiHz`/`DataBits`/`BaudRate`/`WdtTimeout`/`SampleCount`/`CastCount`/`FifoWaterline`/`Speed` 等）。**soft 残留**：bus config 字段私有化+`with_*` builder 延后（config 字段已是受校验 newtype，私有化为低价值 breaking churn；`OutputConfig`/`InputConfig` 已有 `with_*`）。

## C. DMA 完整化 — Hard gate

- [x] ✅ 上层 `start_mem_to_mem(self, src, dst) -> Transfer<'d,T,SRC,DST>`（消费 driver+buffer，`wait()` 归还三者），bound `embedded_dma::{ReadBuffer<Word=u32>,WriteBuffer}`。（`configure_channel` 暂留 public：改 unsafe-private 会破已验证的 HIL 路径而我无法重跑真机复验，故保留并以 `Transfer` 为首选安全层。）
- [x] ✅ 缓存维护折进 transfer（launch `clean_range(src)`、wait `invalidate_range(dst)`，chip-ws63 gated）；`DMA_ALIGN=32` 字节对齐为文档化不变量；`Transfer` early-drop 中止通道（halt+disable）→ UAF guard。**残留**：async `.await` 变体未做（第二个面，HIL-gated）。
- [ ] **真正接进 SPI/I2C/UART**（延后 — 每驱动 DMA-mode 寄存器配置 + 外设握手，体量大且仅真机可验；非 sandbox 可完成）。
- [x] ✅ **验证（编译期部分）**：UAF 在 safe 代码不可表达（`Transfer` 拥有 buffer + abort-on-drop，编译期成立）；**HIL** 新增 `dma_transfer_guard`（走 guard 路径，riscv 编译通过，真机随套件复验）。**残留**：peripheral-DMA 真机路径（依赖上一条）。

## E. drop-to-disable（方案 B）— Hard gate

- [x] ✅ `Wdt`(停表)/`PwmChannel`(关 `pwm_enN`)/`Output`(回 input/高阻) 实现 scoped `Drop`；只关本外设 en 位、不碰共享时钟门。`Timer` 不纳入。
- [x] ✅ 逃生口：`Wdt::into_armed()`/`leak()`→`WatchdogArmed`、`Pwm::into_running()`→`PwmRunning`、`Output::into_latched()`/`into_flex()`→`OutputLatched`（消费 self→零大小 marker；`into_flex` 用 ManuallyDrop 移出字段）。
- [x] ✅ **验证**：**host** marker 零大小单测（寄存器效果无 MMIO 不可 host 测）；**HIL** `wdt_drop_disables_unless_armed`（drop 清 wdt_en、`into_armed` 保持，reset 关闭故安全）。gpio loopback 不受影响（句柄在作用域内）。

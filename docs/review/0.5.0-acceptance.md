## A. 类型化配置收紧（每驱动）— Hard gate

**总不变量**：config 面不接受任何"能写出却被静默 clamp/截断/没接时钟"的值；越界在构造点返回 `None`，不静默处理。

- [ ] **pwm** ✅ 已完成（`PwmPeriod`/`Duty` + 自起时钟，HIL `pwm_configure_and_enable` 绿）。
- [ ] **spi**：`frequency`→`SpiHz`（拒绝超 `[SPI_CLOCK_HZ/0xFFFE, /2]`）、`data_bits`→`DataBits`（实测 DFS 上限）。**scan** 无 D 残留；**host** newtype 边界测试；**HIL** `spi0_loopback` 仍绿。
- [ ] **i2c**：`freq`→`I2cHz`（WS63 对齐 BS2X 的 `Speed`）；SCL 计数不溢出。**host** + **HIL** `i2c0_scl_config` 绿。
- [ ] **uart**：`baudrate`→`BaudRate`（拒绝低端 clamp / 高端 div=0）。**host** + **HIL** `uart0_divider_config` 绿。
- [ ] **i2s**：**type-state** `new_master(非零分频)` / `new_slave()`，零分频 Master 不可表达；两步 `new()+configure()` 合一；自起 CKEN(11/12)。**HIL** `i2s_version_live` 绿。
- [ ] **wdt**：超时越界→`Result`（不静默饱和）。**host** + **HIL** `wdt_configure_saturates_load` 绿。
- [ ] **timer**：`start_us` us→ticks 用 u64 + 越界 `Result`。**host** 溢出测试；**HIL** `timer_counter_advances` 绿。
- [ ] **rtc / gadc / lsadc**：板级/模拟前提（晶振/AFE）→ doc + 守护 + **有界轮询**（无无界 poll）。**grep** 确认无 `while …{}` 无界轮询；rtc/gadc 不上板（文档说明）。
- [ ] **全驱动 scan**：`scan.sh crates/hisi-riscv-hal/src` 无未标注的 A/D 残留。

## B. API 风格清债 — Hard gate（除注明 soft）

- [ ] **错误处理**：8 个 `*Error` 全 `#[non_exhaustive]`；`SpiError::Overflow→ErrorKind::Overrun`；`i2c_v151::I2cError` 接 `embedded_hal::i2c::Error`（BS2X I2C 可移植）。**grep** + **CI**(clippy)。
- [ ] **defmt**：`defmt` optional feature + 错误/config enum 派生 `#[cfg_attr(feature="defmt", derive(defmt::Format))]`；错误 enum 派生归一 `Copy+Eq`。**CI** 加 `--features defmt` job 绿。
- [ ] **文档强制**：`#![warn(missing_docs)]` 开 + backfill 后 `deny`；`get_gpio_mux`→`gpio_mux`；所有 `​```ignore`→`​```no_run`；每个 `pub unsafe fn` 有 `# Safety`，Result/会 panic 的方法有 `# Errors`/`# Panics`。**CI** doc job (`-D warnings`) + **grep**。
- [ ] **所有权**：`peripheral!` 宏加安全 `reborrow(&mut self)`（纯增量）。**host**/**grep**。
- [ ] **trait/sealing**：删 `private.rs` 残留 `PeripheralInput/Output` marker；growth-prone mode enum 视情 `non_exhaustive`；rustdoc 注 `embedded-hal-bus`(SpiDevice) 指引。**grep**。
- [ ] **MSRV**：`[workspace.package].rust-version` 声明 + README 一行政策。**grep** + `cargo +<msrv> check`。
- [ ] **GPIO 双 API**（soft→本波做）：`#[deprecated]` 遗留 `GpioPin<MODE>`、`Flex` 显式 `set_as_input/output`（去隐藏 RMW）、安全 `degrade()`、统一 `ulp_gpio` 命名。**host** + **HIL** `gpio_*` 仍绿。
- [ ] **构造清债**：bus config builder-lite（私有字段/`non_exhaustive` + `with_*`）；裸标量→newtype。**host**。

## C. DMA 完整化 — Hard gate

- [ ] `configure_channel` 收紧为 `unsafe` 私有核心；上层 `start_transfer(self, buf) -> Transfer<'d, BUF>`（消费 driver+buffer，`wait()`/`.await` 归还二者），bound `embedded_dma::{ReadBuffer,WriteBuffer}`。
- [ ] 缓存维护折进 transfer（launch 时 `clean_range(src)`、wait 时 `invalidate_range(dst)`）；32 字节对齐为类型不变量；async 路径同样覆盖（无第二个不一致面）。
- [ ] **真正接进 SPI/I2C/UART**（不只 DMA 自身）。
- [ ] **验证**：use-after-free 在 safe 代码里不可表达（**grep**/编译期）；**HIL** `dma_mem_to_mem` + 至少一个 peripheral-DMA 路径在真机绿。

## D. 中断路由 — Hard gate（跨 hisi-riscv-rt 仓）

- [ ] **hisi-riscv-rt**：`local_isr_dispatch` 接到 `device.x` 具名符号（app 写 `#[interrupt] fn GPIO_INT0()`，不再手抄 `mcause` 解码 / 改 `mtvec`）。
- [ ] HAL 侧 `on_interrupt` 钩子配合；可选 `bind_interrupts!` 式薄 shim。
- [ ] **干掉硬编码 `mcause == 26`**：所有 async/embassy 示例改用 `embassy::ALARM_IRQ`（BS2X 上=53）。**grep** 确认无 `== 26` 字面量。
- [ ] vectored/direct 二选一（删掉未用的一半）。**HIL**/示例在真机中断计数正确。

## E. drop-to-disable（方案 B）— Hard gate

- [ ] `Wdt`/`PwmChannel`/`Output` 实现 scoped `Drop`（停表/关输出/回安全态）；`Drop` 只关本外设 en 位、不碰共享时钟门。
- [ ] 逃生口：`Wdt::into_armed()`/`leak()`、`Pwm::into_running()`（消费 self→无 Drop marker）。`Timer` 不纳入默认 Drop。
- [ ] **验证**：**host** 单测（drop 后寄存器 en 位被清；逃生口保持）；**HIL** 抽查（drop `Wdt` 后不复位）。

## F. chip-feature 去默认 — Hard gate

- [ ] `default = ["rt", "dep:critical-section"]`（**无芯片**）；`compile_error!` 对**零芯片**给清晰提示「启用 chip-ws63 或 chip-bs21 其一」+ `assert_unique_features!` 式守卫。
- [ ] `[package.metadata.docs.rs]`：`default-target = riscv32imfc-unknown-none-elf` + `features=["chip-ws63","rt","async","embassy"]`。
- [ ] **全消费者声明芯片**：`examples/ws63/*`、`examples/bs21`、`examples/bs20`、`tests-hil` 的 hal 依赖加 `features`；**hisi-rs-template** 生成工程声明芯片；**CLAUDE.md** build 命令同步（注明 `cargo check --workspace` 裸跑会报错→用 per-chip 命令）。
- [ ] **验证**：`cargo build -p hisi-riscv-hal`（无 feature）报清晰错；`cargo check -p hisi-riscv-hal --features chip-ws63` 与 `--features chip-bs21` 均绿；docs.rs 渲染（或本地 `cargo doc` 模拟）正确。
